创建hadoop用户

  1. 创建用户hadoop,并且用/bin/bash作为默认shell：
    johnathon@ubuntu16:~$ sudo useradd -m hadoop -s /bin/bash
  2. 为用户hadoop设置密码：
    johnathon@ubuntu16:~$ sudo passwd hadoop
  3. 将用户hadoop加入sudo组：
    johnathon@ubuntu16:~$ sudo adduser hadoop sudo
