Ubuntu16.04 Install Hadoop 2.8.0

发表于 2017-07-02 | 分类于 大数据 | | 阅读次数
| 字数统计 596 (字) | 阅读时长 3 (分)


创建hadoop用户

  1. 创建用户hadoop,并且用/bin/bash作为默认shell：
    johnathon@ubuntu16:~$ sudo useradd -m hadoop -s /bin/bash
  2. 为用户hadoop设置密码：
    johnathon@ubuntu16:~$ sudo passwd hadoop
  3. 将用户hadoop加入sudo组：
    johnathon@ubuntu16:~$ sudo adduser hadoop sudo

更新apt

  • hadoop@ubuntu16:~$ sudo apt-get update

配置ssh免密登陆

  1. 安装ssh服务
    hadoop@ubuntu16:~$ sudo apt-get install ssh
  2. 创建.ssh目录
    hadoop@ubuntu16:~$ cd ~
hadoop@ubuntu16:~$ mkdir .ssh
  3. 生成ssh密钥
    hadoop@ubuntu16:~$ cd .ssh/
hadoop@ubuntu16:~$ ssh-keygen -t rsa
hadoop@ubuntu16:~/.ssh$ cat id_rsa.pub >> authorized_keys
  4. ssh登陆localhost
    hadoop@ubuntu16:~/.ssh$ cd
hadoop@ubuntu16:~$ ssh localhost
  5. 退出ssh登陆
    hadoop@ubuntu16:~$ exit
logout
Connection to localhost closed.

安装配置jdk

  • 参考Ubuntu安装并配置JDK

安装hadoop

  1. 解压hadoop安装文件到相应的目录
    hadoop@ubuntu16:~$ sudo tar -zxf hadoop-2.8.0.tar.gz -C /usr/local
  2. 给hadoop目录重命名
    hadoop@ubuntu16:~$ cd /usr/local
hadoop@ubuntu16:/usr/local$ sudo mv hadoop-2.8.0/ hadoop
  3. 将hadoop目录用户改为hadoop
    hadoop@ubuntu16:/usr/local$ sudo chown -R hadoop hadoop/
  4. 查看hadoop目录
    hadoop@ubuntu16:/usr/local$ cd hadoop/
hadoop@ubuntu16:/usr/local/hadoop$ ls -l

  6. 设置hadoop环境变量
    6.1. 打开hadoop用户下配置文件

    hadoop@ubuntu16:~$ vi .bashrc

    6.2. 编辑文件，添加如下

    #set hadoop env begin
export HADOOP_HOME=/usr/local/hadoop
export PATH=$PATH:$HADOOP_HOME/bin
export PATH=$PATH:$JAVA_HOME/bin
export PATH=$PATH:$HADOOP_HOME/sbin
#set hadoop env end

    6.3. 使文件修改生效

    hadoop@ubuntu16:~$ source .bashrc

    6.4. 查看hadoop版本信息（注意：与上一步骤中查看版本信息的区别）
